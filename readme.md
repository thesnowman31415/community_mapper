### Community Mapper Entwicklungsversion 0.1

Dies ist die erste Entwicklungsversion für den "Community-Mapper". Hier ist es möglich, Events, Personen und Institutionen auf einer Karte (aktuell Weltkarte, später vielleicht eingegrenzt auf Deutschland) zu markieren und so Community-Music öffentlicher zu machen. 

Die restliche Readme wird im Laufe der Entwcklung fortlaufend ergänzt.
